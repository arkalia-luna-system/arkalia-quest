# 🎬 Guide de Démonstration Vidéo - Arkalia Quest

## 📋 Préparation

### 1. Vérifications préalables
- ✅ Python 3.8+ installé
- ✅ ffmpeg installé (`brew install ffmpeg`)
- ✅ Environnement virtuel configuré
- ✅ Serveur fonctionnel sur localhost:5001

### 2. Scripts disponibles
- `launch_demo_video.sh` - Lancement complet automatisé
- `record_demo.sh` - Enregistrement vidéo uniquement
- `demo_script.py` - Test automatique des API
- `demo_guided.py` - Navigation automatique avec Selenium

## 🚀 Lancement rapide

```bash
# Option 1: Lancement complet automatisé
./launch_demo_video.sh

# Option 2: Lancement manuel
./run.sh
# Puis dans un autre terminal:
./record_demo.sh
```

## 🎬 Conseils d'enregistrement

### Qualité vidéo
- **Résolution**: 1920x1080 minimum
- **FPS**: 30 FPS (parfait pour le web)
- **Codec**: H.264 pour compatibilité maximale
- **Qualité**: CRF 18 (excellente qualité)

### Timing
- **Durée totale**: 3-4 minutes maximum
- **Transitions**: 0.5s entre les sections
- **Pauses**: 1-2s pour les commandes importantes
- **Rythme**: Dynamique mais pas précipité

### Audio (optionnel)
- **Musique de fond**: Ambiance cyberpunk/hacker
- **Volume**: 20% pour ne pas couvrir la narration
- **Format**: MP3 128kbps
