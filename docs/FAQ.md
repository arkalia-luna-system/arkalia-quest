# ❓ FAQ - ARKALIA QUEST

> **Questions fréquemment posées et leurs réponses**

## 🎮 **GAMEPLAY**

### **Q: Comment commencer à jouer ?**
**R:** 
1. Va sur https://arkalia-quest-1.onrender.com
2. Clique sur "Commencer l'aventure"
3. Tape `aide` pour voir les commandes
4. Crée ton profil avec `profil`
5. Commence ta première mission avec `mission`

### **Q: Quelles sont les commandes principales ?**
**R:** Les commandes essentielles sont :
- `aide` - Affiche l'aide complète
- `profil` - Gère ton profil
- `mission` - Accède aux missions
- `monde` - Explore le monde d'Arkalia

### **Q: Comment gagner des points et monter en niveau ?**
**R:** 
- **Complète des missions** : 50-100 XP par mission
- **Utilise des commandes** : 5-10 XP par action
- **Découvre des easter eggs** : 25 XP par secret
- **Participe aux défis** : 100+ XP pour les victoires

### **Q: Comment fonctionne le système de badges ?**
**R:** Les badges se débloquent automatiquement :
- **Badges de progression** : Niveaux et missions
- **Badges secrets** : Easter eggs et actions spéciales
- **Badges de défi** : Victoires en compétition

## 🤖 **IA LUNA**

### **Q: Qui est LUNA ?**
**R:** LUNA est ton **IA complice** avec une personnalité unique :
- **Rebelle** : Insultes amicales et vocabulaire gaming
- **Intelligente** : Analyse tes actions et s'adapte
- **Humoristique** : Memes dynamiques et réponses personnalisées

### **Q: Pourquoi LUNA m'insulte ?**
**R:** Les "insultes" de LUNA sont **amicables et humoristiques** ! C'est sa personnalité rebelle qui :
- Te motive avec de l'humour décalé
- Crée une ambiance gaming authentique
- Te fait rire avec des memes

### **Q: Comment LUNA s'adapte à mon style de jeu ?**
**R:** LUNA analyse :
- **Tes actions** : Commandes utilisées, missions choisies
- **Ton niveau** : Progression et compétences
- **Ton style** : Créatif, analytique, social, équilibré
- **Ton historique** : Actions précédentes

## 🎯 **MISSIONS ET PROGRESSION**

### **Q: Combien y a-t-il de missions ?**
**R:** Il y a **6 missions complètes** :
1. **Introduction** - Découvrir Arkalia Quest
2. **Premier Contact** - Rencontrer LUNA
3. **Défi Hacker** - Résoudre des énigmes
4. **Cyber Escape** - Échapper à La Corp
5. **Boss Final** - Affronter le boss final
6. **Tutoriel Interactif** - Apprendre les bases

### **Q: Que se passe-t-il si je rate une mission urgente ?**
**R:** Les missions urgentes ont des **timers** :
- **10 secondes** pour sauver ton historique
- **15 secondes** pour retrouver tes memes
- **20 secondes** pour défier La Corp

Si tu rates : **Pas de panique !** Tu peux recommencer et tu ne perds pas de progression.

### **Q: Comment débloquer les easter eggs ?**
**R:** Essaie ces commandes secrètes :
- `luna_dance` - Fais danser LUNA
- `boss_final` - Affronte le boss final
- `easter_egg_1337` - Découvre un secret
- `challenge_corp` - Défie La Corp

## ⚔️ **DÉFIS SOCIAUX**

### **Q: Comment participer aux défis temps réel ?**
**R:** 
1. Tape `challenge_corp` pour créer un défi
2. Partage le **Room ID** avec tes amis
3. Ils rejoignent avec le même Room ID
4. Le défi commence automatiquement

### **Q: Que sont les "rooms" ?**
**R:** Les rooms sont des **salles de compétition** :
- **Création** : Crée ta propre room
- **Jointure** : Rejoins une room existante
- **Chat** : Communique avec les autres
- **Actions** : Synchronise tes mouvements

### **Q: Comment gagner des défis ?**
**R:** 
- **Réagis vite** : Les défis ont des timers
- **Utilise ta créativité** : Solutions originales
- **Collabore** : Travaille avec ton équipe
- **Pratique** : Plus tu joues, mieux tu deviens

## 🔧 **TECHNIQUE ET DÉPANNAGE**

### **Q: Le site ne se charge pas, que faire ?**
**R:** 
1. **Vérifie ta connexion** Internet
2. **Rafraîchis la page** (F5)
3. **Vide le cache** du navigateur
4. **Essaie un autre navigateur** (Chrome, Firefox, Safari)

### **Q: Les commandes ne marchent pas**
**R:** 
1. **Vérifie l'orthographe** de la commande
2. **Utilise `aide`** pour voir les commandes disponibles
3. **Redémarre le jeu** si nécessaire
4. **Contacte le support** si le problème persiste

### **Q: J'ai perdu mon profil**
**R:** 
1. **Vérifie la sauvegarde** automatique
2. **Recrée ton profil** si nécessaire
3. **Contacte le support** si problème persiste

### **Q: Le jeu est lent**
**R:** 
1. **Ferme d'autres onglets** du navigateur
2. **Vérifie ta connexion** Internet
3. **Rafraîchis la page**
4. **Essaie un autre navigateur**
